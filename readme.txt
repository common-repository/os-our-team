== Description ==

How well you present your team members on your website is crucial in building customer trust and confidence. So we present you an excellent WordPress plugin, that will help you display your team members in a cool, attractive style. This plugin offers three different layout options as shown in the screenshots. Please take a look at them in detail below:

Shortcode

We have created three types of display methods

1. Classic layout For classic layout use this shortcode
[os-our-team columns="3" order="id" order="desc" display="os-classic"]

2. Muse layout For classic layout use this shortcode
[os-our-team columns="3" order="id" order="desc" display="os-muse"]

3. Grid Layout For classic layout use this shortcode
[os-our-team columns="3" order="id" order="desc" display="os-grid"]. For better display for "Grid Layout" please use columns="2" in shortcodes.

== Installation ==

Minimum requirements:    
    
 Wordpress 4.3+    
 PHP 5.4.x    
 MySQL 5.4.x+  

Installation from within backend:

1. In plugin manager, click "Add New"
2. Search for "os-our-team"
3. Click "install"
4. Activate the plugin once it is installed
5. Click on "Our Team" from the left menu and follow the instructions to complete the installation

FTP upload installation method:

1. Upload the `os-our-team` folder to the `/wp-content/plugins/` directory using your FTP client
2. Activate the plugin through the 'Plugins' menu
3. Go to the Our Team menu, finish the installation and manage blog appearance

Zip upload installation method (make sure uploads up to 5MB are allowed for your hosting):

1. Login to your WordPress site administrator panel and head over the 'Plugins' menu  
2. Click 'Add New'  
3. Choose the 'Upload' option
4. Click **Choose file** (**Browse**) and select the os-our-team.*.zip file.   
5. Click **Install Now** button.    
6. Once it is complete, activate the plugin.   
7. Go to the Our Team menu, finish the installation and manage blog appearance
